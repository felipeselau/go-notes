📘 Introdução ao Markdown

Markdown é uma linguagem de marcação leve que permite formatar texto de maneira simples.

Tópicos abordados

  • Títulos
  • Ênfase
  • Listas
  • Links

Exemplo de código

package main

import "fmt"

func main() {
    fmt.Println("Olá, Markdown!")
}
